Git Basics

Git thinks about its data different then other VCSs. Git stores and thinks about its data differently then other VCSs. Git uses snapshots to file their data, unlike other programs that use lists to store their data.

Git Repository Basic Commands

In this lesson we learned the basic commands that are used in Git. Like using the git config command to configure a repository. Or the commit command to commit the repository. All of these commands are very important in storing your data in Git.

Git Repository Recording Changes

When making changes to your index.html, you have to add them to your Git using the add command. This will store your changes. You then commit your project again, and push your project. You then refresh your Git page and your changes are stored.
